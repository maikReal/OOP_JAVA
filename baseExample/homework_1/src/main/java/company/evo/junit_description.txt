@BeforeClass -> можем объявлять различные объекты и прочее. Запускается эта часть перед запуском тестов. Запускается только один раз, когда запускается класс тестов
@AfterClass -> запускаем эту часть после завершения тестов. Запускается только один раз, когда запускается класс тестов
@Before -> запускается каждый раз перед началом теста
@After -> запускается каждый раз перед началом теста
@Test -> здесь пишем сами тесты
